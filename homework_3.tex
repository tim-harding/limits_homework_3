\section*{2.4.8}

\subsection*{Problem}

Prove that $\lim_{x \to x_0} x = x_0$.

\subsection*{Solution}

$\forall \epsilon > 0,\ \exists \delta = \epsilon,\ \forall x : 0 < \abs{x - x_0} < \delta,\ \abs{x - x_0} < \epsilon$
